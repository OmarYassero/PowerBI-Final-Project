# PowerBI-Final-Project
Power BI analysis of customer churn for Databel Telecom. Analyzed 6,687 records to identify drivers for a 26.86% churn rate. Features 4 interactive dashboards exploring demographics, contract types, and usage patterns using custom DAX. Transformed raw data into insights to help stakeholders improve retention
